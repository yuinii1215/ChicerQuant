    让我们从股票的收益率开始跟随数学的脚步吧！
    收益率的计算大有玄机，需要考量以下两个方面：
    1、印花税
    2、复利

    印花税 是股票交易时需要向监管部门缴纳的税款，我国暂行的规定是卖出时征收千分之一
    单期收益率为R，则其对等的连续复利收益率应为(1＋R)的自然对数，即ln(1＋R)。
    之所以选择连续复利收益率，主要有一下两个原因：
    1、连续复利的公式以e为底数，可积可导指数可加，计算方便
    2、此处的模型抽象并不太影响现实结论
    3、它的概率分布较接近于正态分布
    接近正态分布又有什么好处呢？请接着探寻.

    -predict

    一种历史资料延伸预测，也称历史引伸预测法。是以时间数列所能反映的社会经济现象的发展过程和规律性，进行引伸外推，预测其发展趋势的方法。
    时间序列是一种完全不同的事物,它源于我们生活中的直觉，即:
    1、股票的明天的收益会受今天，昨天、、历史的影响，并且时间间隔越长，影响也越小
    2、股票收益是不可能完全被预测的
    因此，在构建时序模型之前，我们要先检验：过去的历史的确对未来有影响
    这里我们选取了一下两个统计量作为检验依据：自相关系数和偏自相关系数

    pacf

    其中，标线为取95%置信区间时的临界值，当相关系数的绝对值大于此值时，说明该期的影响较为明显
    同时，我们还引入平稳性的概念，作为时间序列预测的基本假设。即：时间序列的行为并不随时间改变。
    体现在数学上的弱平稳为：
    ① the mean function μ(t) is constant over time(均值函数是常数函数)
    ② γ(t, t − k) = γ(0, k) for all times t and lags k(协方差函数仅与时间差相关)
    我们采用增广迪基-福勒检验(Augmented Dickey-Fuller test)来检验平稳性

    该假设的原假设是目标序列是平稳的，因此，只要检验值p小于5%即可认为符合条件

    Step 3&4

    -predict

    经过了这么多的准备，终于到了应用ARMA模型的时候
    ARMA模型(auto regressive moving average model)自回归滑动平均模型，模型参量法高分辨率谱分析方法之一。这种方法是研究平稳随机过程有理谱的典型方法，适用于很大一类实际问题。
    它比AR模型法与MA模型法有较精确的谱估计及较优良的谱分辨率性能，但其参数估算比较繁琐。
    但是在计算机面前，这些都很容易，我们借助python的统计包直接给予参数并模拟，这里省略了模型的细节

    拿到了预测的结果也不要太激动，我们还有最后的小尾巴——残差
    举一个形象的例子，这些历史数据就好比生产原料，ARMA模型相当于生产工艺，评价生产工艺的好坏要有很多方面，而残差则相当于留下的废料。
    从某种意义上说，废料中可利用的信息越少，模型提取的信息就越多。
    好在统计学中给予了我们白噪声来类比。下面我们就需要验证残差趋近于白噪声。
    白噪声其实就是一组符合正态分布的数据，是“自然”存在的。我们选取了Q-Q图、正态分布检验、LBQ检验、Durbin-Watson检验来说明。

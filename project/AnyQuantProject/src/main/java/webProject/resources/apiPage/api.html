什么是ChicerQuant

ChicerQuant是一个高效易用的股票策略学习助手，主要提供用户在线编写量化策略和运行策略等功能，旨在为所有量化爱好者 提供一个具有丰富数据的量化平台，chicerQuant提供了如下几个功能：

  • 策略：在每个用户独立的环境中，用户可以编写自己的策略并提交到系统运行回测。
  • 社区：所有用户可以在社区中发布、分享自己的策略。
  • 数据：我们提供了策略依赖的数据，目前提供了自2006年来的A股数据，后面如果有需要，我们会加入更多的股票数据。

输入的股票名均值股票代号

日期格式为xxxx-xx-xx


股票数据

  • getStockByNameService(username,name,date)

    输入:股票名和日期

    输出: 操作是否成功的信息以及该日该股票的属性--各种表格项信息

  • getStockAmongDateService(name, startdate, enddate)

    输入: 股票名和日期区间

    输出: 操作是否成功的信息以及该日区间内的每日该股票的属性--各种表格项信息

  • getAllStocksService()

    输入: 无

    输出: 当日所有股票属性--各种表格项信息

  • getAllStockNamesService()

    输入:无

    输出:得到所有的股票名


大盘数据

  • getBenchMarkByNameService(name, date)

    输入: 大盘名和日期

    输出: 操作是否成功的信息以及该日该大盘的属性--各种表格项信息

  • getBenchMarkAmongDateService(name, startdate, enddate)

    输入: 大盘名和日期区间

    输出: 操作是否成功的信息以及该区间内每日该大盘的属性--各种表格项信息


图表数据

  • getDayLineService(name, startdate, enddate)

    输入: 股票名或大盘名和日期区间

    输出: 操作是否成功的信息以及该区间内每日该股票的图表有关数据信息

  • getWeekLineService(name, startdate,enddate)

    输入: 股票名或大盘名和日期区间

    输出: 操作是否成功的信息以及该区间内该股票每周一的图表有关数据信息

  • getMonthLineService(name, startdate, enddate)

    输入: 股票名或大盘名和日期区间

    输出: 操作是否成功的信息以及该区间内该股票每月第一天的图表有关数据信息

  • getPolyAmongDateService(name, startdate, enddate)


行业数据

  • getAllIndustriesService()

    输入: 无

    输出: 操作是否成功的信息以及所有行业的名称

  • getStocksByIndustryService(industry_name)

    输入: 行业名称

    输出: 操作是否成功的信息以及该行业的所有股票名(最新)

  • getIndustryService(industry_name,date)

    输入: 行业名称和日期

    输出: 操作是否成功的信息以及该日该行业的所有属性--表格项信息
